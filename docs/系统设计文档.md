## 1. 系统概述

### 1.1 项目背景

随着工业化、城镇化快速推进，空气质量问题日益突出。及时准确地监测和预测空气质量，对政府决策、公众出行、环境保护具有重要意义。

### 1.2 项目目标

本系统旨在构建一个基于Hive的城市空气质量数据仓库，实现多维度空气质量分析，并运用机器学习算法进行短期预测，为环境管理提供决策支持。

### 1.3 系统功能

| 功能模块 | 描述 |
|----------|------|
| 数据采集 | 自动采集空气质量监测数据和气象数据 |
| 数据处理 | ETL清洗、转换、加载至数据仓库 |
| 多维分析 | 时空分布、相关性、趋势分析 |
| 预测预警 | AQI预测、等级预警 |
| 可视化展示 | 实时看板、专题地图、趋势图表 |

### 1.4 系统特点

- **海量数据处理**：基于Hadoop/Hive处理GB级历史数据
- **多维分析**：支持时空、周期性、多因素相关性分析
- **智能预测**：集成多种机器学习模型，提供24-72小时预测
- **实时可视化**：Web端实时展示，支持多终端访问

### 4.2 模块详细设计

| 功能 | 描述 | 数据来源 |

## 5. 数据库设计

### 5.1 数据仓库分层

```
┌─────────────────────────────────────────────────────────────┐
│  ADS (Application Data Store) - 应用数据层                  │
│  用途：支持前端查询的预计算结果                              │
│  特点：数据量小、查询频繁、更新及时                          │
├─────────────────────────────────────────────────────────────┤
│  DWS (Data Warehouse Summary) - 数据汇总层                   │
│  用途：宽表汇总、多维分析                                    │
│  特点：按主题聚合、轻度汇总                                  │
├─────────────────────────────────────────────────────────────┤
│  DWD (Data Warehouse Detail) - 明细数据层                   │
│  用途：明细事实数据、维度退化                                │
│  特点：原子数据、最细粒度                                    │
├─────────────────────────────────────────────────────────────┤
│  ODS (Operational Data Store) - 操作数据层                  │
│  用途：原始数据缓存、过渡层                                  │
│  特点：保持原始格式、未清洗                                  │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 主要表结构

详见 `air-spark-analysis/src/main/resources/hive-sql/01_database_and_tables.sql`

| 表名 | 类型 | 说明 |
|------|------|------|
| ods_air_quality_raw | ODS | 空气质量原始数据 |
| ods_weather_raw | ODS | 气象数据原始表 |
| dwd_air_quality_dt | DWD | 空气质量明细表 |
| dwd_weather_dt | DWD | 气象数据明细表 |
| dws_station_hour | DWS | 站点小时汇总表 |
| dws_district_day | DWS | 区县日汇总表 |
| ads_realtime_aqi | ADS | 实时AQI数据表 |
| ads_aqi_prediction | ADS | 预测结果表 |
| dim_station | 维度 | 站点维度表 |
| dim_time | 维度 | 时间维度表 |

### 5.3 分区策略

| 表类型 | 分区字段 | 分区粒度 |
|--------|----------|----------|
| ODS | dt | 每日 |
| DWD | year/month/day | 每日 |
| DWS | year/month | 每月 |
| ADS | 无分区/静态分区 | - |

## 7. 预测模型设计

### 7.1 特征工程

```python
# 时间特征
时间特征 = [hour, day_of_week, month, is_workday, is_holiday]

# 滞后特征
滞后特征 = [pm25_lag_1, pm25_lag_6, pm25_lag_12, pm25_lag_24,
            pm10_lag_1, pm10_lag_6, pm10_lag_12,
            o3_lag_1, o3_lag_6, o3_lag_12]

# 同期特征
同期特征 = [pm25_same_hour_yesterday, pm25_same_hour_lastweek,
            pm25_same_day_lastweek]

# 统计特征
统计特征 = [pm25_rolling_6h_avg, pm25_rolling_24h_avg,
            pm10_rolling_6h_avg, o3_rolling_6h_avg]

# 气象特征
气象特征 = [temperature, humidity, wind_speed, pressure, visibility]
```

### 7.2 模型对比

| 模型 | 适用场景 | MAE | RMSE | 准确率 | 训练时间 |
|------|----------|-----|------|--------|----------|
| RandomForest | 通用 | ~12 | ~18 | ~78% | 中 |
| XGBoost | 回归/分类 | ~10 | ~15 | ~80% | 中 |
| LSTM | 时序预测 | ~8 | ~12 | ~82% | 长 |
| CNN-LSTM | 复杂时序 | ~7 | ~11 | ~84% | 很长 |
| 集成模型 | 最优性能 | ~6 | ~10 | ~86% | 很长 |

### 7.3 模型评估指标

- **回归指标**: MAE、RMSE、MAPE、R²
- **分类指标**: 准确率、精确率、召回率、F1
- **业务指标**: AQI等级准确率（≥75%）
