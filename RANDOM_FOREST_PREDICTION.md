# 随机森林空气质量预测功能

## 功能概述

基于**5天历史数据**，使用**随机森林算法**预测次日（第6天）的空气质量，包括：
- AQI指数
- 六项污染物数据（PM2.5、PM10、SO2、NO2、CO、O3）
- 特征重要性分析
- 预测置信度

## 算法特点

### 随机森林模型参数
- **树的数量**: 100棵决策树
- **最大深度**: 10层
- **特征采样**: 每次分裂随机选择4个特征
- **Bootstrap采样**: 有放回抽样构建每棵树的训练集

### 考虑的特征因素
1. **历史AQI趋势** - 最近几天的AQI变化趋势
2. **PM2.5浓度** - 细颗粒物浓度
3. **PM10浓度** - 可吸入颗粒物浓度
4. **SO2浓度** - 二氧化硫浓度
5. **NO2浓度** - 二氧化氮浓度
6. **CO浓度** - 一氧化碳浓度
7. **O3浓度** - 臭氧浓度
8. **季节因素** - 春夏秋冬的季节影响

## 使用步骤

### 1. 访问预测页面
- 打开前端页面：`http://localhost:3000`
- 点击左侧菜单「**随机森林预测**」

### 2. 配置预测参数
- **选择站点**：中原区、金水区、二七区、惠济区、郑东新区
- **选择历史数据结束日期**：预测将基于该日期前5天的数据

### 3. 加载历史数据
点击「**加载历史数据**」按钮，系统将自动从CSV文件加载最近5天的历史数据。

### 4. 执行预测
点击「**使用随机森林预测次日数据**」按钮，算法将：
1. 构建100棵决策树的随机森林
2. 基于历史数据训练模型
3. 预测次日AQI和六项污染物
4. 计算特征重要性
5. 输出预测置信度

### 5. 查看结果
预测结果包括：
- **AQI主卡片**：显示预测AQI值、等级、首要污染物
- **六项污染物卡片**：详细展示各项污染物的预测值
- **特征重要性分析**：显示哪些因素对预测影响最大
- **置信度**：表示预测结果的可靠程度

## 预测原理

### 1. 数据预处理
- 解析历史数据中的AQI和六项污染物
- 提取季节特征（根据日期判断春夏秋冬）
- 计算AQI趋势（相邻日期的变化）

### 2. 随机森林构建
```
对于每棵树（共100棵）:
  1. Bootstrap采样：从历史数据中有放回抽取样本
  2. 递归构建决策树:
     - 随机选择4个特征
     - 寻找最佳分裂点（最小化方差）
     - 分裂数据为左右子集
     - 递归构建子树直到满足停止条件
```

### 3. 预测过程
```
输入：最近一天的数据作为特征向量
输出：
  1. 每棵树给出独立预测
  2. 取100棵树的平均值作为最终AQI预测
  3. 基于AQI比例推算六项污染物
  4. 计算预测置信度（方差越小置信度越高）
```

### 4. 特征重要性计算
- 统计每个特征在所有树中作为分裂节点的次数
- 归一化处理得到重要性权重
- 排序展示前5个重要特征

## 注意事项

1. **数据要求**
   - 需要至少5天的连续历史数据
   - 数据质量影响预测准确性

2. **预测限制**
   - 预测结果仅供参考，实际空气质量受多种因素影响
   - 置信度表示模型对预测结果的确定程度

3. **算法特点**
   - 随机森林通过集成学习降低过拟合风险
   - 对异常值不敏感
   - 能处理特征间的非线性关系

4. **性能考虑**
   - 100棵树的构建在普通服务器上约需1-2秒
   - 预测结果是确定性的（相同输入产生相同输出）
